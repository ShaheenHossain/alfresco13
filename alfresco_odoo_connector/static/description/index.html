ALFRESCO - ODOO CONNECTOR

  • This application will allow you to connect ODOO with your ALFRESCO repository and upload all types of documents into Alfresco repository directly from Odoo.
  • You can manage all the documents directly from Odoo itself
  • It is very easy and simple to use

    • Features of ALFRESCO - ODOO CONNECTOR

    You can update document in Alfresco Repository through ODOO from this modules

    1. SALE ORDER
    2. PURCHASE ORDER
    3. INVOICES & BILLS
    4. STOCK PRODUCTION LOT
    5. MAINTENANCE & EQUIPMENTS
    6. PROJECT TASK
    7. CONTACTS
    8. EMPLOYEE

    STEPS TO CONNECT ODOO WITH ALFRESCO:

    1. Go to Alfresco Menu
    2. The user needs to authenticate with the Alfresco Repository by using their login credentials of Alfresco
      1. Username of Alfresco
      2. Password of Alfresco
      3. Alfresco Share URL
    3. After clicking on the GET TICKET button the user will get a token which means that user has logged in successfully into alfresco repository.

    UPLOADING DOCUMENT INTO ALFRESCO REPOSITORY FROM ODOO

    1. After the authentication has been done, the user can upload any type of document from the Odoo modules into Alfresco repository.
      1. Documents : From here the user can upload all the documents into Alfresco repo
      2. Attachments : This gives you count of documents available from that particular folder.
      3. Refresh Repository : You can use this button to refresh the document list after uploading or deleting documents.
      4. Create Folder : This button will create a folder into Alfresco Repository
      5. Upload : This button will allow you to upload multiple documents into Alfresco Repository.
    2. Click on Folder button
    3. Click on Upload button
    4. Pop up will be open, from this pop you have to attach document and click on Upload File
    5. Once document is uploaded in Alfresco repository then click on Refresh button
    6. Then you can see numbers of uploaded documents in Alfresco repository in Smart button
